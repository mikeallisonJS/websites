# Ecteon — Sr. Programmer

## What I worked on

Designed and developed the Contraxx contract-management platform —
Ecteon's flagship product. Owned a major Delphi-to-.NET port, built out
the security model to meet enterprise-grade compliance, and architected
a web application that mirrored and then expanded on the Win32 client
using AJAX technologies.

## Wins

- Ported the existing Delphi codebase to C# / ASP.NET.
- Built a web application using AJAX technologies that mirrored and
  expanded on the existing Win32 client's functionality.
- Implemented strict security policies surpassing OWASP / HIPAA / HITECH
  requirements.
- Maintained a high level of cross-browser compatibility including
  legacy IE support.
- Added Single Sign-On support.
- Created comprehensive web services for customer and AJAX consumption.
- Designed a custom serializable data layer for scalability and
  uniformity across the product.
- Created event-driven modular external libraries so client-specific
  code could be added cleanly in C# or VB.
- Cleanly separated data, business, and view logic, and stood up
  common-use libraries and domain structure.

## Tech

ASP.NET, C#, VB.NET, JavaScript, CSS, SQL Server, Oracle, Delphi,
Ajax.NET, AjaxControlToolkit, jQuery, jQueryUI, Knockout, LINQ.
