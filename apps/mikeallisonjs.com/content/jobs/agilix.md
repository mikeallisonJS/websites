# Agilix — Lead UI Developer

## What I worked on

EdTech platform building white-label LMS and high-stakes assessment tooling
for downstream education customers. I led the UI for a distributed
online/offline assessment application on the MEAN stack, targeting mobile,
desktop, and browser environments — students could continue testing without
a network and sync back when reconnected. Also contributed to Agilix's core
`xli-ng` framework and maintained the existing Backbone / XLI.JS / DLAP
enterprise UI.

## Wins

- Designed and shipped a distributed online/offline application for
  high-stakes assessments using a MEAN stack (Mongo, Express, Angular,
  Node) packaged for mobile, desktop (Node-Webkit), and browser.
- Added accessibility support to the `xli-ng` framework.
- Designed C# interface middleware so third-party clients could consume
  partial applications.
- Maintained and improved the existing enterprise Backbone / XLI.JS / DLAP
  UI application.
- Contributed to core `xli-ng` framework development.

## Tech

CoffeeScript, Angular, JavaScript, CSS, MongoDB, Java, C#, Backbone,
Node.js, Express, Socket.io, jQuery, Node-Webkit, Grunt, Gulp.
